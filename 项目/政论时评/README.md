# 政论时评

> 郜老师风格政论时评字幕系统

---

## 📋 项目简介

**风格定位**: 抖音政论时评（郜老师风格）
**视频规格**: 9:16竖屏（450x800px）
**字幕系统**: V6专业版（UI Pro Max优化）
**核心特色**:
- 动态数字滚动（Counter动画）
- 智能背景图匹配（5个系列100张）
- News/Media专业配色
- 精准时间轴同步

## 🔧 技术栈

### TTS生成
- **引擎**: IndexTTS2（本地零样本克隆）
- **音色**: Kevin（主讲者）
- **速度**: 1.5倍（atempo=1.5）
- **采样率**: 22050Hz
- **费用**: 0元（本地生成）

### 字幕系统
- **V6版本**: UI Pro Max优化
- **动画引擎**: CSS Animation + JavaScript
- **Counter**: 数字滚动动画（40步）
- **Timeline**: 精确到0.01秒

### 视频合成
- **工具**: FFmpeg + Python
- **帧率**: 30fps
- **编码**: H.264

## 📞 常见问题

**Q: 如何制作新剧集？**
A: 查看[快速开始.md](./快速开始.md)完整流程

**Q: Counter数字不动怎么办？**
A: 检查timeline中counter的触发时机配置

**Q: 文字太长溢出？**
A: 在剧本JSON中手动指定`font_size`缩小字号

**Q: 背景图不满意？**
A: 在scene中添加`bg_override`手动指定图片
