## 技术细节

### 时间轴压缩算法
```javascript
// 压缩比例
const ratio = 37.25 / 42; // 0.8869

// 所有时间点乘以ratio
timeline.forEach(scene => {
  scene[1] *= ratio; // start time
  scene[2] *= ratio; // duration
  scene[3].forEach(anim => {
    anim[0] *= ratio; // animation delay
  });
});
```

### 动效系统
- **fade**: 淡入 + 向上平移（translateY）
- **pop**: 放大弹出（scale）
- **counter**: 数字滚动动画（40步）

### 背景切换
- 每个场景对应一张gallery图片
- 1秒渐变过渡（opacity transition）
- 暗化处理（brightness 0.45）
