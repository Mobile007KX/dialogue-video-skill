## 📊 场景类型参考

| 场景类型 | 说明 | 推荐背景图系列 | 文字样式建议 |
|---------|------|---------------|-------------|
| `hook` | 金句、开场 | gallery_color（鲜艳） | block-yellow标签 |
| `fact` | 事实、叙事 | gallery_composition（构图） | white普通文字 |
| `data` | 数据、统计 | gallery_focus（聚焦） | 数字用display大号 + counter |
| `contrast` | 对比、反转 | gallery_lighting（光影） | 黄白对比 |
| `emphasis` | 强调、重点 | gallery_focus（聚焦） | block-accent橙色标签 |
| `transition` | 过渡、转折 | gallery_composition（构图） | white-stroke描边 |
| `key` | 核心观点 | gallery_focus（聚焦） | yellow大字 + stroke-heavy |
| `closing` | 收尾、总结 | gallery_whitespace（留白） | dim暗淡文字 |

## 🎨 文字样式参考

### 色块标签
- `block-yellow` - 黄色标签（金句、核心概念）
- `block-primary` - 蓝色标签（分论点：第一、第二）
- `block-accent` - 橙色标签（强调、转折）

### 文字颜色
- `white` - 白色普通文字
- `white-stroke` - 白色文字+描边（重要语句）
- `yellow` - 黄色高亮（数据、关键词）
- `red` - 红色警示（负面数据）
- `light` - 浅色次要文字
- `dim` - 暗淡辅助文字（收尾）

### 字号
- `t1` (52px) - 一级标题
- `t2` (40px) - 二级标题、正文
- `display` (80px) - 超大数字

### 动画
- `fade` - 淡入上移（常规文字）
- `pop` - 弹出放大（重点、标签）

## ⚙️ 高级配置（可选）

### 自定义背景图

如果想手动指定背景图，在scene中添加：

```json
{
  "id": "s1",
  "type": "hook",
  "bg_override": "gallery_color_15.jpg",  // 手动指定
  "lines": [...]
}
```

### 自定义字号

如果文字太长，需要缩小字号：

```json
{
  "text": "这是一段很长的文字需要缩小字号才能放得下",
  "style": "white",
  "anim": "fade",
  "font_size": 32  // 手动指定字号（覆盖默认）
}
```
