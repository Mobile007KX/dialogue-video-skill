# 在线试听 & 体验链接汇总

## 直接可用的在线工具

### 1. CosyVoice（阿里开源，免费）

| 版本 | 链接 | 特点 |
|------|------|------|
| **CosyVoice 2.0** | https://www.modelscope.cn/studios/iic/CosyVoice2-0.5B | 最新版，支持流式推理 |
| CosyVoice 300M | https://www.modelscope.cn/studios/iic/CosyVoice-300M | 稳定版 |

**支持的方言**：粤语、四川话、东北话、陕西话、山西话、上海话、天津话、山东话、宁夏话、甘肃话、郑州话、长沙话等 18+ 种

**使用方法**：
1. 打开链接
2. 上传3秒音频 或 选择预设音色
3. 输入要说的文字
4. 可输入指令如"用东北话说"、"用开心的语气说"
5. 点击生成

## 推荐体验顺序

### 橘猫大哥（东北土豪）

1. **最快**：打开剪映 → 文本朗读 → 选「东北老铁」
2. **效果好**：去 CosyVoice 2.0 → 输入文字 → 指令填"用东北话说这句话，语气痞一点"
3. **最自然**：找一段东北大哥说话的视频 → 提取音频 → 上传到 Reecho 克隆

### 贵宾小姐（名媛嗲嗲）

1. **最快**：打开剪映 → 文本朗读 → 选「台湾甜妹」
2. **效果好**：去讯飞在线TTS → 选「讯飞玉儿」（台湾女声）
3. **最自然**：找林志玲/虞书欣说话的视频 → 提取音频 → 上传到 Reecho 克隆

## 示例：CosyVoice 指令写法

```
# 橘猫大哥
文本：整挺好，今儿个哥带你逛街，LV随便挑
指令：用东北话说这句话，语气要拽一点，像社会大哥

# 贵宾小姐
文本：哎呀，人家要那个包包嘛
指令：用台湾腔说这句话，语气嗲嗲的，像在撒娇
```
