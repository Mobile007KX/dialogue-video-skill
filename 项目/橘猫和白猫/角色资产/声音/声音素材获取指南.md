## 方案一：直接使用AI配音工具（最快）

这些工具自带东北腔、台湾腔等预设音色，可以直接生成：

### 1. 剪映（推荐，免费）
- **下载**：手机App或电脑版
- **路径**：文本朗读 → 选择主播
- **东北腔**：东北老铁、东北大哥
- **台湾腔**：台湾甜妹
- **优点**：免费、操作简单、音质好

### 2. 配音火火
- **地址**：微信小程序搜索"配音火火"
- **音色**：200+抖音热门发音人
- **方言**：东北话、台湾话、四川话等
- **优点**：情绪可调（开心、悲伤、兴奋）

### 3. 快配酱
- **地址**：微信小程序搜索"快配酱"
- **东北腔**：东北-铁柱、东北-欣姐、东北-半吨
- **优点**：语速可调、支持下载MP3

### 4. 讯飞在线TTS
- **地址**：https://www.xfyun.cn/services/online_tts
- **方言**：11种（东北话、台湾普通话、四川话等）
- **发音人**：100+
- **优点**：专业级、可商用

## 方案二：声音克隆（效果最好）

上传一段带口音的真人音频，AI学习后可以用这个声音读任何文字。

### 1. CosyVoice（阿里开源，免费）
- **地址**：https://github.com/FunAudioLLM/CosyVoice
- **在线体验**：https://www.modelscope.cn/studios/iic/CosyVoice-300M
- **克隆要求**：3秒音频即可
- **方言支持**：粤语、四川话、上海话、天津话、武汉话等
- **优点**：完全免费、本地运行、效果好

### 2. 海螺AI (Hailuo AI)
- **地址**：https://hailuoai.video/
- **克隆要求**：10秒音频
- **优点**：免费、中文支持好

### 3. Fish Audio
- **地址**：https://fish.audio/
- **特点**：70+语言、1000+声音
- **优点**：有免费额度

### 4. Reecho睿声
- **地址**：https://www.reecho.ai/
- **克隆要求**：3-10秒
- **优点**：国内首选、效果自然

### 5. TransDuck
- **地址**：https://www.transduck.com/
- **方言口音**：粤语、四川话、上海话、郑州话、长沙话、天津话、南京话、武汉话
- **优点**：专注方言配音

## 方案三：从视频提取声音素材

### 步骤
1. 找到合适的视频（抖音/B站/快手）
2. 提取音频
3. 截取干净的人声片段（10-30秒）
4. 上传到声音克隆平台

### 音频提取工具

| 工具 | 用途 | 地址 |
|------|------|------|
| cobalt | 抖音/B站视频下载 | https://cobalt.tools/ |
| yt-dlp | 命令行下载工具 | `brew install yt-dlp` |
| 剪映 | 提取音频 | 导入视频 → 导出音频 |
| Vocal Remover | 人声分离 | https://vocalremover.org/ |

### B站音频提取命令
```bash
# 安装yt-dlp
brew install yt-dlp

# 下载B站视频的音频
yt-dlp -x --audio-format mp3 "B站视频链接"
```

5. 输入台词生成配音

## 注意事项

1. **版权问题**：克隆明星/网红声音仅供个人娱乐，商用需获得授权
2. **音频质量**：用于克隆的音频尽量清晰、无背景音乐
3. **时长要求**：大多数克隆平台需要10-30秒的纯人声
