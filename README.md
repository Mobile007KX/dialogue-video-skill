## 概述

**适用场景**：纯对话推动剧情的短剧
- 相声/脱口秀
- 情侣/夫妻日常
- 吐槽/怼人

**核心流程**：
```
定装照 → 角色设定 → 素材生成 → 剧本创作 → TTS → 合成出片
```

## 快速开始

### 1. 准备素材

每个场景需要3个文件：
```
├── 起始帧.jpg          # 定装照（两角色同框）
├── A说话.mp4           # A说话，B闭嘴听
└── B说话.mp4           # B说话，A闭嘴听
```

### 2. 生成视频素材的关键

⚠️ **Prompt必须明确：一个说话，另一个闭嘴**

```
✅ 正确：
   Character A is TALKING with mouth movements.
   Character B is LISTENING SILENTLY, mouth firmly CLOSED.

❌ 错误：
   Two characters having a conversation.  ← 两个都会动嘴
```

### 3. 写剧本

```markdown
【A】台词1
【B】台词2
【A】台词3
...
```

### 4. 生成TTS + 合成

```bash
python 工具/合成脚本.py
```

## 核心技巧

详见 `制作指南/分镜与运镜指南.md`

| 技巧 | 说明 |
|------|------|
| 角色不离开画面 | 两个角色始终在画面中 |
| 固定机位 | 背景动，角色不出画面 |
| 首尾相连 | 视频可无缝循环，长台词不跳帧 |
| 一说一听 | 明确指定谁说话谁闭嘴 |
